## 🎭 Full Meeting Simulation Script

**Tình huống:** Sprint Review Meeting — 35 phút (bạn sẽ ghi âm phiên bản rút gọn 7 phút)

**Participants (bạn đóng tất cả các vai):**
- Bạn = QC Manager (facilitator + QC report)
- PM = Sarah (muốn release Friday)
- Dev Lead = Minh (có 2 blockers chưa xong)

---

**[OPEN — 1 phút]**
```
YOU: "Alright, let's get started. Thanks everyone for joining. 
Today's sprint review has three items: QC update, 
release decision, and planning for next sprint.
I want to be out of here in 30 minutes — let's make it count."
```

**[QC UPDATE — 1.5 phút]**
```
YOU: "Let me walk you through the QC status.
This sprint we ran 148 test cases. 
131 passed, 12 failed, 5 still in progress.
Pass rate: 88.5% — up from 80% last sprint. Good improvement.

Two open blockers: a critical bug in the checkout flow 
and a performance issue under high load.
Both are logged. Dev team is aware."
```

**[DISCUSSION — 2 phút]**
```
PM (you playing PM): "Okay — can we still release Friday? 
We promised the client."

YOU (as QC): "I hear you, Sarah. And I respect the commitment to the client.
But I have to be transparent: the checkout bug is Severity 1.
If we ship with that, we risk 30% of users hitting a payment failure.
That's a worse outcome than a short delay.

My recommendation: 24-hour delay to Saturday morning.
Dev fixes the bug tonight. QC does a targeted regression tomorrow morning.
We release Saturday noon — client gets it over the weekend."

PM: "What about the performance issue?"

YOU: "That one is Severity 2 — I can live with shipping that 
with a logged ticket and a fix in the next sprint.
But not the checkout bug."
```

**[DECISION — 1 phút]**
```
PM: "Okay. You've convinced me. Saturday it is.
Minh, can your team fix the checkout bug by tonight?"

DEV LEAD (you): "Yes — we'll have a fix by 10pm."

YOU (as QC): "Perfect. I'll run regression first thing tomorrow morning.
And Sarah — can you notify the client about the 24-hour delay?"

PM: "On it."
```

**[CLOSE — 1 phút]**
```
YOU: "Great. Let me capture the action items.
Dev team: fix checkout bug by tonight 10pm.
QC: regression run tomorrow 8am.
Sarah: client notification today.
Next sprint planning — I'll send the agenda by EOD.

Thanks everyone — productive session. Talk soon."
```
