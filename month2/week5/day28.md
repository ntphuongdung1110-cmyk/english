# Day 28 — Virtual Meeting Etiquette
**Thứ Năm | Tháng 2, Tuần 5**
**Chủ đề:** Họp online — ngôn ngữ và kỹ năng đặc thù cho video calls

---

## 📚 Từ Vựng Hôm Nay — Virtual Meeting Language

| Tình Huống | Câu Nói Tự Nhiên |
|---|---|
| Bắt đầu khi có người chưa vào | "Let's give it another minute for others to join." |
| Confirm nghe rõ | "Can everyone hear me okay?" / "Audio good?" |
| Bị lag / technical issue | "Sorry — my connection cut out. Could you repeat that?" |
| Mute nhầm | "Oh sorry — I was on mute. What I was saying was..." |
| Hỏi ai đó lên tiếng | "I think [Name] was about to say something — go ahead." |
| Chia sẻ màn hình | "Let me pull up the report — give me one second." |
| Kết thúc call | "Thanks everyone — great session. I'll drop the notes in Slack." |
| Kết nối kém | "I'm going to turn off my camera — connection issues. But I'm here." |

---

## 🗣️ Mẫu Câu — Situations Unique to Video Calls

### Khi bị interrupt vì technical issue
```
"Sorry to cut in — I think there might be an echo. 
Could everyone except [speaker] go on mute?"
```

### Khi muốn nói nhưng nhiều người đang nói
```
"I'll jump in quickly when there's a pause — 
just want to add one thing."
```

### Khi cần đọc / chia sẻ tài liệu
```
"I'm going to share my screen for this next part — 
just a moment while I pull it up.
Can everyone see the report now?"
```

## 🎧 Shadowing Script

```
"Hey, can everyone hear me okay? Great.

So before we dive in — I see we're still waiting on two people.
Let's give it another minute.

Okay, looks like we have most people. Let's get started 
and we'll catch [Name] up when they join.

Quick housekeeping: if you're not speaking, 
please go on mute — there's some background noise.
And if you want to jump in, just unmute and go for it — 
no need to raise your hand.

I'm going to share my screen for the QC dashboard — 
give me one second.
Can everyone see this? Good."
```
