## 📚 Từ Vựng Hôm Nay — Advanced Meeting Openers

### Mở Meeting Theo Tình Huống

**Formal / Leadership meeting:**
```
"Good morning everyone. Thank you for making time — 
I know schedules are tight. Let's make this count."

"I appreciate everyone being here. 
I want to keep us focused so we can make the most of our time."
```

**Casual / Team standup:**
```
"Alright, let's dive in. Quick check — 
is everyone ready to get started?"

"Morning everyone! Before we begin — 
any urgent blockers I should know about?"
```

**Crisis / Urgent meeting:**
```
"Thanks for jumping on so quickly. 
We have a situation that needs a fast decision — 
I'll get right to it."

"I'll skip the small talk — we have a critical issue 
and I want to make sure we leave this call with a clear action plan."
```

### Setting a Productive Tone
```
"I want this to be a focused session — 
so I'll be keeping us on track."

"My goal for this meeting is [outcome] — 
let's see if we can get there in [time]."

"I'll send pre-reads ahead of time so we can use this time for discussion."

"Ground rules: please hold questions until [point], 
and let's aim to finish by [time]."
```

## 🗣️ Mẫu Câu — Setting Agenda Professionally

### Pattern 1: Agenda có thứ tự ưu tiên
```
"Today's agenda has [X] items. I've prioritized them by urgency.
First — [most important]. This needs a decision today.
Second — [item]. This is for visibility, no decision needed.
Third — if time allows — [optional item]."
```

### Pattern 2: Time-boxing agenda
```
"I'm going to time-box each item so we stay on track.
[Item 1] — 10 minutes.
[Item 2] — 15 minutes.
[Item 3] — 5 minutes for wrap-up and action items."
```

### Pattern 3: Outcome-focused opening
```
"By end of this meeting, I need us to:
One — decide on [decision].
Two — align on [plan].
Three — agree on next steps.
Everything else can wait."
```

## 🎧 Shadowing Script

```
"Good morning everyone. Thanks for being here 
and for coming in on time — I appreciate that.

Today's meeting has three objectives. 
First, I want to give you a QC status update 
on this sprint — that'll take about 5 minutes.
Second, I need a decision on the release date — 
we'll spend 10 minutes on that.
And third, if time allows, 
I'd like to briefly discuss our automation roadmap.

Ground rule for today: 
I'll be the one keeping us on track with time. 
If we go off topic, I'll flag it and suggest 
we take it offline. Is everyone okay with that?

Great. Let's get started."
```
