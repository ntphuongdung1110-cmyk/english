## 🧠 Podcast vs TED — Điểm Khác Biệt

| | TED Talks | Podcasts |
|---|---|---|
| Tốc độ | Vừa phải, chuẩn bị kỹ | Nhanh hơn, tự nhiên hơn |
| Vocabulary | Academic, formal | Casual, nhiều idiom/slang |
| Structure | Rõ ràng, có script | Thường là conversational |
| Độ khó nghe | ★★★☆ | ★★★★ |
| Benefit | Phát âm chuẩn, cấu trúc tốt | Native speed, daily English |

→ Lý tưởng: kết hợp cả hai mỗi tuần.
