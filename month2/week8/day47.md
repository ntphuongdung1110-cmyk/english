## 📋 Quick Review — Tháng 2 Skill Areas

### Area 1: Meeting Skills (Tuần 5)
```
□ Mở meeting với clear outcome
□ Time-box agenda
□ Facilitate discussion — kéo mọi người tham gia
□ Capture action items: who / what / when
□ Virtual meeting language
```

### Area 2: Feedback Skills (Tuần 6)
```
□ Positive feedback: specific + impact + forward
□ SBI model: Situation + Behavior + Impact
□ Receiving feedback: acknowledge + clarify + commit
□ Performance review language: meets/exceeds expectations
□ Difficult feedback: direct + supportive + clear expectations
```

### Area 3: Presentation Skills (Tuần 7)
```
□ PREP method: Point → Reason → Example → Point
□ Signposting: "Moving on to..." / "As you can see..."
□ Describing trends: "jumped from X to Y" / "dropped by X%"
□ Data storytelling: "The story behind the numbers is..."
□ Q&A handling: confident + honest + buy time when needed
```

### Area 4: Advanced Listening (Tuần 8)
```
□ TED Talk: hiểu >65% không phụ đề
□ Podcast: hiểu >55% native speed
□ Shadowing: bắt chước rhythm + stress + intonation
□ Speaking speed: đẩy lên >100 từ/phút
```
