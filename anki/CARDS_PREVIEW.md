## Deck 1 — QC Vocabulary (45 cards)

### Testing Types

| Thuật ngữ | Nghĩa | Ví dụ câu |
|---|---|---|
| **smoke test** | Test nhanh xem build có chạy được không | Run a smoke test before full regression. |
| **sanity test** | Test sau fix để verify nhanh | Quick sanity check after the hotfix. |
| **regression test** | Test lại để đảm bảo không có lỗi mới | Full regression takes 2 days. |
| **exploratory test** | Test sáng tạo, không theo script | Allocate time for exploratory testing. |
| **UAT** | User Acceptance Testing | Client UAT starts Monday. |
| **E2E test** | End-to-end — test toàn luồng | E2E covers the full checkout flow. |
| **integration test** | Test tích hợp giữa các module | Integration tests caught 3 API issues. |
| **performance test** | Test hiệu suất (load, stress, spike) | Performance testing under 1000 concurrent users. |
| **security test** | Test bảo mật | Security test flagged a SQL injection risk. |
| **usability test** | Test trải nghiệm người dùng | Usability test with 5 real users next week. |

### Defect & Bug Language

| Thuật ngữ | Nghĩa | Ví dụ câu |
|---|---|---|
| **defect / bug** | Lỗi phần mềm | We logged 12 defects this sprint. |
| **severity** | Mức độ nghiêm trọng (S1–S4) | This is Severity 1 — it crashes the app. |
| **priority** | Mức độ ưu tiên xử lý | High priority — fix before Friday. |
| **blocker** | Lỗi chặn release | We have 2 blocker bugs open. |
| **reproduce** | Tái hiện lỗi | I can reproduce this on iOS 17, step 3. |
| **root cause** | Nguyên nhân gốc rễ của lỗi | Root cause: null pointer exception in auth. |
| **regression** | Lỗi cũ quay lại sau fix mới | This is a regression from last sprint. |
| **workaround** | Cách xử lý tạm thời | There's a workaround but we need a proper fix. |
| **hotfix** | Fix khẩn cấp sau release | We pushed a hotfix at 2am. |
| **false positive** | Test báo lỗi nhưng thực ra không có | 3 of those were false positives. |
| **flaky test** | Test không ổn định — đôi khi pass đôi khi fail | This test is flaky — fails 30% of the time. |

### Process & Workflow

| Thuật ngữ | Nghĩa | Ví dụ câu |
|---|---|---|
| **test plan** | Kế hoạch kiểm thử | I'll have the test plan ready by Monday. |
| **test case** | Kịch bản kiểm thử | We have 200 test cases for this feature. |
| **test suite** | Bộ các test cases | The regression suite covers 85% of core features. |
| **test coverage** | % code / feature được test | We're at 78% test coverage. |
| **acceptance criteria** | Tiêu chí chấp nhận của story | Story is missing clear acceptance criteria. |
| **definition of done** | Tiêu chí hoàn thành | QC sign-off is part of our definition of done. |
| **sign-off** | Phê duyệt / chấp nhận kết quả test | QC sign-off required before go-live. |
| **go-live** | Phát hành sản phẩm ra production | Go-live is scheduled for Friday 6pm. |
| **rollback** | Quay lại phiên bản cũ | We had to rollback after the prod incident. |
| **deployment** | Triển khai lên môi trường | Deployment to staging is complete. |

### Agile & Team

| Thuật ngữ | Nghĩa | Ví dụ câu |
|---|---|---|
| **sprint** | Chu kỳ phát triển 1–2 tuần | We're in sprint 12 right now. |
| **backlog** | Danh sách công việc chưa làm | This is in the backlog for next sprint. |
| **standup** | Họp ngắn hằng ngày | I raise blockers in the daily standup. |
| **retrospective** | Họp nhìn lại sau sprint | Retrospective is Friday afternoon. |
| **shift left** | Đưa QC vào sớm hơn trong SDLC | We're shifting left — QC from day one. |
| **CI/CD** | Tích hợp & triển khai liên tục | Tests run automatically in our CI/CD pipeline. |
| **automation coverage** | % test đã được tự động hóa | Automation coverage is currently 40%. |

### Reporting & Metrics

| Thuật ngữ | Nghĩa | Ví dụ câu |
|---|---|---|
| **bug escape rate** | % lỗi lọt qua QC ra production | Our bug escape rate is 2% this quarter. |
| **pass rate** | % test cases pass | 88% pass rate this sprint. |
| **MTTD** | Mean Time To Detect — thời gian trung bình phát hiện lỗi | MTTD decreased from 3 days to 1 day. |
| **MTTR** | Mean Time To Resolve — thời gian trung bình sửa lỗi | MTTR is currently 2.5 days. |
| **quality gate** | Ngưỡng chất lượng tối thiểu để tiến | We failed the quality gate — can't proceed. |
| **risk-based testing** | Test ưu tiên theo rủi ro | We use risk-based testing to prioritize. |
| **test summary report** | Báo cáo tổng kết kiểm thử | I'll send the test summary report by EOD. |

## Deck 2 — Advanced Expressions (38 cards)

### Agreement

| Expression | Nghĩa | Dùng khi |
|---|---|---|
| **"That's a fair point."** | Điểm đó có lý | Đồng ý một phần |
| **"I can see where you're coming from."** | Tôi hiểu quan điểm của bạn | Không đồng ý nhưng vẫn lịch sự |
| **"You're not wrong about that."** | Bạn không sai về điều đó | Softer than "you're right" |
| **"I hadn't thought about it that way."** | Tôi chưa nghĩ theo hướng đó | Mở ra cuộc thảo luận |
| **"You make a compelling case."** | Bạn có lập luận thuyết phục | Khi bị thuyết phục |
| **"I'm coming around to that view."** | Tôi đang dần chấp nhận quan điểm đó | Từ từ thay đổi ý kiến |

### Expressing Uncertainty

| Expression | Nghĩa | Dùng khi |
|---|---|---|
| **"It's hard to say with certainty, but..."** | Khó nói chắc, nhưng... | Professional hedging |
| **"I'd take that with a grain of salt."** | Tôi sẽ xem xét kỹ điều đó | Skeptical nhưng lịch sự |
| **"Off the top of my head..."** | Không suy nghĩ kỹ thì... | Trả lời nhanh, chưa chắc |
| **"I'm not 100% sure, but my gut says..."** | Không chắc 100%, nhưng tôi cảm thấy... | Chia sẻ intuition |
| **"That's still up in the air."** | Vẫn chưa quyết định | Chủ đề chưa được chốt |
| **"We're in uncharted territory here."** | Chúng ta đang đi vào vùng chưa biết | Tình huống mới, chưa có kinh nghiệm |

### Buying Time

| Expression | Nghĩa | Dùng khi |
|---|---|---|
| **"Give me a second to think through this."** | Cho tôi một chút để suy nghĩ | Thay vì im lặng awkward |

| **"Can I come back to you on that?"** | Tôi có thể trả lời bạn sau không? | Cần thêm thời gian |

### Emphasizing

| Expression | Nghĩa | Dùng khi |
|---|---|---|
| **"And this is key..."** | Và điều này rất quan trọng | Highlight điểm chính |
| **"Let me be crystal clear about this."** | Để tôi nói rõ ràng về điều này | Tránh hiểu nhầm |
| **"I can't stress this enough."** | Tôi không thể nhấn mạnh đủ điều này | Very strong emphasis |
| **"The bottom line is simple."** | Kết luận rất đơn giản | Trước khi nói điểm chốt |
| **"This is the crux of the issue."** | Đây là cốt lõi của vấn đề | Root cause / core issue |

### Transitions

|---|---|---|

| **"The flip side is..."** | Mặt kia của vấn đề là... | Present counterpoint |

### Leadership Language
