## 📊 Bài Kiểm Tra Cuối Tuần 2

### Test 1: Flash Quiz (không nhìn bảng)

Nói to tiếng Anh cho từng tình huống:

```
1. Bắt đầu nêu vấn đề trong meeting:
   → "I'd like to _______ a concern about..."

2. Phản đối lịch sự khi PM muốn skip testing:
   → "I _______ you, but I'd _______ back on that."

3. Kết thúc câu chuyện ngắn:
   → "Long story _______, we fixed it and shipped on time."

4. Đồng ý và bổ sung:
   → "That makes sense. On _______ of that, we should also..."

5. Dự đoán rủi ro:
   → "I _______ some issues with the new API integration."

6. Kết thúc small talk:
   → "Anyway, I'll let you _______ back to it."
```

**Đáp án:** flag / hear / push / top / anticipate / get

### Test 2: Nghe (3 phút)

Bật 1 video Brian mới (chưa xem):
```
□ Hiểu được: ___% (mục tiêu tuần 2: >50%)
□ Filler words nghe được: _______________
□ 3 từ/cụm mới bắt được: _______________
```

### Test 3: Ghi Âm — Full Scenario (5 phút)

> **"You're in a weekly team meeting. The PM asks for a QC update. Share:**
> 1. Last week's results (storytelling)
> 2. A problem you found and your recommendation (problem-solving)
> 3. Plans for next sprint (future plans)
> 4. End with a small talk comment to wrap up naturally."
