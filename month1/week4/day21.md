## 🗣️ Mẫu Câu Hôm Nay

### Pattern 1: Mô tả tuần bận rộn (casual)
```
"This week has been [adjective], honestly. 
We've got [situation], so [what's happening].
But we're getting through it."
→ This week has been pretty hectic, honestly. 
   We've got a release on Friday, so everyone's in crunch mode.
   But we're getting through it.
```

### Pattern 2: Nói về stress mà không kêu ca
```
"It's been a lot lately, but [positive spin / coping strategy]."
→ It's been a lot lately, but I've been trying to take short breaks 
   between tasks — it really helps me stay focused.
```

### Pattern 3: Nói về work-life balance
```
"Something I've been working on is [habit/boundary].
I realized that [insight], so I started [change]."
→ Something I've been working on is disconnecting after 7pm. 
   I realized that checking Slack late at night was affecting my sleep, 
   so I started putting my phone on silent after dinner.
```

### Pattern 4: Raising workload concern professionally
```
"I want to flag that the team's workload right now 
is [description]. If we keep this pace, I'm concerned about [risk].
I'd like to discuss [potential solution]."
→ I want to flag that the team's workload right now is unsustainable. 
   If we keep this pace through the next sprint, I'm concerned about burnout. 
   I'd like to discuss reprioritizing some tasks.
```

## 🎧 Shadowing Script — Work Stress Conversation

```
"Honestly? It's been a tough few weeks.
We've had two back-to-back releases, 
and the team has really been pushing hard.

I think we're getting to a point where 
the pace is starting to take a toll — 
people are getting tired, mistakes are happening. 
And I get it — pressure from above is real.

But I also know that burnout doesn't help anyone. 
A burned-out tester misses bugs. 
And that's worse for everyone in the long run.

So I've had a conversation with my team 
about setting some boundaries. 
No urgent pings after 8pm unless it's truly critical.
Taking a proper lunch break.
Small things, but they make a difference.

At the end of the day, sustainable pace 
leads to better quality — and that's always been my north star."
```
