# Master List — Sentence Patterns
**Tham khảo nhanh. Học thuộc từng nhóm theo tháng.**

---

## THÁNG 1 — Foundation Patterns

### Nhóm 1: Giới Thiệu & Mô Tả Bản Thân
```
I've been [verb-ing] for [X] years.
My main responsibility is [verb-ing / to verb].
I'm currently [verb-ing] / in charge of / responsible for [noun].
One thing I pride myself on is [noun/verb-ing].
Outside of work, I enjoy [verb-ing].
I have a background in [field].
I specialize in [area].
```

### Nhóm 2: Diễn Đạt Ý Kiến
```
From my perspective, [opinion].
The way I see it, [opinion].
I tend to think [clause].
As far as I'm concerned, [opinion].
It seems to me that [clause].
I strongly believe [clause].
Honestly, [frank statement].
I understand [other view], but personally I think [your view].
```

### Nhóm 3: Kể Chuyện
```
It all started when [event].
At the time, [situation].
What happened was [event]. So I [action].
Long story short, [brief conclusion].
Looking back, [reflection].
In hindsight, [what I'd do differently].
What I took away from that was [lesson].
```

### Nhóm 4: Đồng Ý / Phản Đối
```
Absolutely. / Exactly. / Fair enough.
I see your point, but [concern].
I'd push back on that — [reason].
I hear you, though I have some concerns about [specific].
I agree to some extent, but [different angle].
You make a good point. I'd also add that [your point].
With all due respect, I think [different view].
Can we agree to [compromise]?
```

### Nhóm 5: Meeting Language
```
I'd like to flag a [severity] issue with [area].
The root cause appears to be [cause].
My recommendation would be to [action].
Do we have an ETA on [fix/decision]?
As of [time], [X] out of [Y] test cases have passed.
Going forward, we should [preventive action].
I'm open to other suggestions.
```

## THÁNG 2 — Professional Patterns

### Nhóm 6: Giving Feedback
```
I think [positive], and one thing to consider is [improvement].
I noticed [observation] — have you thought about [suggestion]?
What worked well was [positive]. What I'd suggest improving is [area].
I appreciate [effort], and I think we can take it further by [action].
To be direct — [honest feedback] — and here's how I think we can fix it.
```

### Nhóm 7: Asking for Clarification
```
Just to make sure I understand — are you saying [restatement]?
Could you help me understand the rationale behind [decision]?
When you say [term], do you mean [interpretation A] or [interpretation B]?
I want to make sure I'm aligned — so the expectation is [your understanding], correct?
Could you give me an example of what that would look like?
```

### Nhóm 8: Presenting Results
```
I'd like to walk you through [topic].
Let me start with [starting point], then move on to [next].
As you can see from [chart/data], [observation].
This is significant because [reason].
In summary, [key takeaway].
Based on these results, I'd recommend [action].
Are there any questions before I move on?
```

### Nhóm 9: Running a Meeting
```
Let's get started — today's agenda has [X] items.
Before we dive in, does anyone have any urgent updates?
I'd like to hear from [name/team] on [topic].
Let's keep this one in the parking lot for now.
We're running short on time — let's move on.
To summarize what we've agreed on: [summary].
Next steps: [person] will [action] by [deadline].
```

### Nhóm 10: Email-to-Speech (Spoken Equivalents)
```
Written: "Please find attached..."
Spoken: "I've attached [file] — let me know if you need anything else."

Written: "As per my previous email..."
Spoken: "Like I mentioned before..." / "To recap what I said..."

Written: "Please advise."
Spoken: "What do you think?" / "Let me know how you'd like to handle this."

Written: "I hope this email finds you well."
Spoken: "How have you been?" / "Hey, hope you're doing well."
```

## THÁNG 3 — Fluency Patterns

### Nhóm 11: Expressing Uncertainty (mà không nghe yếu)
```
I'm not 100% sure, but I think [tentative view].
Don't quote me on this, but [uncertain claim].
I'd have to double-check, but my understanding is [belief].
That's a good question — let me get back to you on that.
Off the top of my head, [quick answer], but I'll confirm.
```

### Nhóm 12: Handling Difficult Conversations
```
I want to have a candid conversation about [topic].
This isn't easy to say, but [honest statement].
I think we need to address [issue] before it becomes a bigger problem.
I'm not trying to point fingers — I just want to understand what happened.
How can we make sure this doesn't happen again?
I appreciate you hearing me out.
```

### Nhóm 13: Thinking Aloud / Buying Time
```
That's something I've been thinking about a lot lately.
Let me think through this out loud for a second.
On one hand [point A], on the other hand [point B].
So here's my thinking — [explain reasoning step by step].
Actually, now that you mention it...
I hadn't thought about it that way before.
```

## THÁNG 4–6 — Advanced Patterns

### Nhóm 14: Negotiating & Persuading
```
I think we can both agree that [common ground].
What if we approached it this way — [proposal].
The way I see it, [your view] would actually benefit [them] because [reason].
I'm flexible on [less important point], but I do need [important point].
Is there a middle ground we can find here?
```

### Nhóm 15: Leadership Communication
```
I want to make sure the team has everything they need.
My door is always open if anyone has concerns.
I trust your judgment on this — just keep me posted.
Let's celebrate this win — the team worked hard.
I take responsibility for [issue] — here's what we're doing to fix it.
What do you need from me to make this happen?
```
