# Phrasal Verbs — Công Việc & QC
**Học 3–5 cụm mỗi tuần. Ôn lại mỗi thứ Hai.**

---

## Nhóm 1: Meeting & Communication

| Phrasal Verb | Nghĩa | Ví Dụ Câu |
|---|---|---|
| **bring up** | đề cập, nêu ra | I need to bring up a concern about the release. |
| **follow up on** | theo dõi tiến độ | I'll follow up on that bug report tomorrow. |
| **loop in** | kéo ai vào vòng thông tin | Let me loop in the dev lead before we decide. |
| **go over** | xem xét, review | Can we go over the test results together? |
| **run through** | trình bày nhanh / xem qua | Can you run through the bug list for me? |
| **wrap up** | kết thúc | Let's wrap up — we're running low on time. |
| **kick off** | bắt đầu | We'll kick off testing on Monday morning. |
| **check in with** | kiểm tra / hỏi thăm | I'll check in with the team after lunch. |
| **circle back** | quay lại chủ đề sau | Let's circle back to that after the demo. |
| **take offline** | bàn riêng, không trong meeting | Let's take this offline — too detailed for now. |

## Nhóm 2: Problem-Solving & Decision

| Phrasal Verb | Nghĩa | Ví Dụ Câu |
|---|---|---|
| **look into** | điều tra, tìm hiểu | I'll look into the root cause and report back. |
| **figure out** | tìm ra / giải quyết | We need to figure out why this keeps failing. |
| **narrow down** | thu hẹp phạm vi | Let's narrow down the test scope to critical paths. |
| **rule out** | loại trừ | We've ruled out a database issue — it's in the frontend. |
| **come up with** | nghĩ ra | The team came up with a good workaround. |
| **fall back on** | dùng phương án dự phòng | If automation fails, we'll fall back on manual testing. |
| **push back** | phản đối / trì hoãn | I need to push back on this deadline — it's not realistic. |
| **sign off on** | phê duyệt | Who needs to sign off on this before we release? |
| **roll out** | triển khai | We're rolling out the fix to production tonight. |
| **iron out** | giải quyết chi tiết nhỏ | Let's iron out the details in tomorrow's sync. |

## Nhóm 3: Progress & Work Flow

| Phrasal Verb | Nghĩa | Ví Dụ Câu |
|---|---|---|
| **keep up with** | theo kịp | It's hard to keep up with all the changes. |
| **catch up on** | cập nhật lại | I need to catch up on the test reports I missed. |
| **fall behind** | bị tụt hậu | We're falling behind on the test schedule. |
| **get bogged down** | bị sa lầy | Don't get bogged down in low-priority bugs. |
| **cut down on** | giảm bớt | We need to cut down on manual regression time. |
| **step up** | tăng cường / đảm nhận thêm | The team needs to step up before the release. |
| **burn out** | kiệt sức | The team is burning out from the crunch. |
| **ramp up** | tăng tốc dần | We're ramping up testing in the final sprint. |
| **hand off** | chuyển giao | I'll hand off the test cases to the automation team. |
| **pick up** | tiếp nối / nhận việc | Can you pick up where Sarah left off? |

## Nhóm 4: Escalation & Reporting

| Phrasal Verb | Nghĩa | Ví Dụ Câu |
|---|---|---|
| **escalate to** | báo cáo lên cấp trên | I need to escalate this to the PM today. |
| **flag up** | báo hiệu vấn đề | I want to flag up a risk before we proceed. |
| **point out** | chỉ ra | The tester pointed out 3 critical issues. |
| **lay out** | trình bày rõ ràng | Let me lay out the risks for you. |
| **sum up** | tóm tắt | To sum up — 2 blockers, 1 critical, 3 minor. |
| **break down** | phân tích chi tiết | Let me break down the test results by module. |
| **account for** | giải thích / chiếm tỉ lệ | These 5 bugs account for 80% of our risk. |
| **draw attention to** | thu hút sự chú ý | I want to draw attention to this edge case. |
| **set aside** | gác lại / để riêng | Let's set aside the low-priority items for now. |
| **put off** | trì hoãn | We shouldn't put off fixing this known issue. |

## Cách Học Hiệu Quả

```
1. Học 5 cụm / tuần (không cố nhồi hết)
2. Thêm vào Anki với VÍ DỤ câu thực tế của bạn
3. Dùng ít nhất 2 cụm trong ghi âm buổi sáng
4. Thứ Bảy: Ôn lại 20 cụm đã học
```
